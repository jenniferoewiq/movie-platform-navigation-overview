# movie-platform-navigation-overview
该项目围绕在线电影与电视剧网站的导航整体概览展开整理，对首页导航层级逻辑、分类标签体系以及播放页信息结构进行系统归档。通过结构化文字记录建立资料库，用于分析不同影视平台在导航设计与页面规划方面的变化趋势。仓库仅保存公开页面资料。
